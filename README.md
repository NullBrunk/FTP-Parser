# ftp-parser

This is a simple python3 parser. His goal is to find FTP creds in .pcap files. 

# How i install ?

```
# dependecies :
pip3 install termcolor scapy

# git cloning :
git clone https://github.com/NullBrunk/ftp-parser/
cd ftp-parser && chmod +x ftp

# help
./ftp -h

```
