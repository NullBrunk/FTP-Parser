# FTP parser

Parse .pcap file searching for FTP credentials


# How to install ?

```bash
# dependecies :
pip3 install termcolor scapy

# git cloning :
git clone https://github.com/NullBrunk/FTP-Parser/
cd FTP-Parser && chmod +x ftp

# help
./ftp -h

```
